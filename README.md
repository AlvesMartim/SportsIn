Lancement de l'application
---------------------------------------------------------------------------------------------------------------------------------
Pour lancer le projet éxecutez  "./gradlew bootRun" (sous linux) dans votre console.
Attendez un peu et il sera écrit que l'app est en lancée en local ( en général sur le port 8080 ).
Ne vous inquitéz pas si le chargement est bloqué à 80% c'est normal.
Accèdez à localhost:8080 sur votre navigateur pour accéder à l'application
